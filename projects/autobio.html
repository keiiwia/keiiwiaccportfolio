Autobiographical Game

The autobiographical game is a game revolving around my childhood, childhood nightmares,
and how they interact with the place that I call home.

The objective of this game is to collect items, which in turn, influences the main character's outer appearance. As the game progresses,
the main character continues to age, extending beyond even my current age to display what I hope the future looks like.

This is currently the greyboxed/whiteboxed version of the game, as the assets are still in creation.
